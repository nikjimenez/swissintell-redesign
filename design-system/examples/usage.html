Spacing tokens in action

Every margin, padding and gap in this page uses a --space-* token. No hardcoded pixel values.

The section padding above is var(--space-16), the demo title margin is var(--space-6), and the card grid gap is var(--space-5). Change the token once, and the entire site updates.
